## 3. 고도화 로드맵 (Tier별 · 영향/난이도/데이터)

### Tier 1 — 즉시 효과, 전부 무료데이터로 가능 (다음 스프린트 권장)
1. **애널리스트 컨센서스 카드** ★ — 목표주가(평균/고/저), 투자의견(매수/보유/매도), 애널리스트 수. `yfinance .info(targetMeanPrice, recommendationKey, numberOfAnalystOpinions)` + `.recommendations`. → *Seeking Alpha 느낌, 난이도 低, 임팩트 高*
2. **관심종목 워치리스트** — localStorage 저장, 미니 시세 테이블(현재가·등락·미니 스파크라인). → *토스 "내 주식"/TV 워치리스트, 무인증, 난이도 中*
3. **종목 비교 뷰** — 2~3종목 정규화 가격 차트 + 지표 비교 표(수익률·PER·RSI·시총). `/api/compare?tickers=`. → *Koyfin/TV, 난이도 中, 임팩트 高*
4. **차트 강화** — 기간 선택(1M·3M·6M·1Y·5Y), RSI 서브차트, MA 토글. → *난이도 中*
5. **다크모드 + 디자인 토큰 정리** — 고대비 색 토큰(상승/하락), 핵심수치 타이포 스케일, 여백. → *2026 트렌드, 난이도 低*
6. **시장 현황 홈** — 주요 지수(KOSPI `^KS11`·코스닥 `^KQ11`·S&P `^GSPC`·나스닥 `^IXIC`)·환율(USDKRW=X)·미니 차트. → *토스 홈, 난이도 中*

### Tier 2 — 제품 깊이
7. **실적 캘린더·배당** — 다음 실적일, 배당락/배당수익률. `yfinance .calendar/.dividends`.
8. **간이 스크리너** — KRX/S&P500 종목 리스트를 **사전계산(배치)** → 필터(PER·등락률·RSI·시총). *Finviz 라이트. 실시간 스캔은 무료론 불가 → 일배치 + 정직 표기*
9. **포트폴리오 트래커** — 보유·평단 입력(localStorage) → 평가손익·비중. *토스 "내 주식"*
10. **AI 고도화** — 종목 비교/요약 전용 프롬프트, **답변 스트리밍**(SSE), "왜 이 신호인지" 설명, 뉴스 LLM 요약.

### Tier 3 — 고급
11. **가격 알림** — 클라이언트 폴링 기반(목표가 도달 시 브라우저 알림).
12. **섹터 히트맵** — 보유/관심 종목 등락 히트맵.
13. **뉴스 감성 분석** — LLM로 뉴스 긍/부정 태깅.

## 4. 디자인 고도화 (2026 핀테크 기준)
- **디자인 시스템 도입**: 색 토큰(상승=빨강/하락=파랑 KR 기본 + 토글), 타이포 스케일(핵심수치 bold·대형), 8px 그리드 여백, 카드 일관성.
- **다크/라이트 모드** 토글(고대비).
- **레이아웃 재편**: 토스식 홈(시장 현황 + 워치리스트) → 종목 상세 → 비교. 사이드 AI 챗 유지.
- **글랜서빌리티**: 미니 스파크라인, 핵심 KPI 상단 고정, 모바일 1탭 동선.
- **신뢰 신호**: 모든 수치에 출처/기준일(이미 강점) — 더 부각.
- **주의(이전 교훈)**: 과한 애니메이션/페이드는 깜빡임 유발 → 절제. (웹폰트 @import 대신 시스템 폰트)

## 5. 권장 "다음 스프린트" (1주, 증권사 느낌으로 도약)
Tier 1에서 **5개 묶음**:
`애널리스트 컨센서스` + `워치리스트` + `종목 비교 뷰` + `차트 기간/지표 강화` + `다크모드`
→ 백엔드: `get_analyst`, `/api/compare` 추가 · 프론트: 워치리스트(localStorage)·비교 페이지·테마 토글·차트 옵션.

## 5-b. 디자인 리디자인 (벤치마킹 → 채택 방향)
조사: Robinhood(흑백+단일 네온 액센트·모듈러·미니멀), Webull(레드/블랙), Bloomberg 터미널(다크·모노스페이스 숫자·앰버), 2026 대시보드(다크우선+CSS변수 테마), 데이터 시각화 원칙(대비≥4.5:1, **색+형태 병행**, 색 7~10 제한, 산세리프).

**채택: 미니멀 라이트 (Robinhood/토스 계열)** — 디자인 시스템:
- **색**: 배경 `#f7f8fa`/카드 `#fff`, 브랜드 액센트 **민트 `#00c39a`**(등락색과 충돌 없음), 상승 `#f04452`(빨강)·하락 `#3182f6`(파랑) = 한국 관례. 등락은 **색+화살표(▲▼)** 병행으로 색각 접근성 확보.
- **타이포**: Pretendard/Inter, 핵심 가격 `2.5rem` Bold, 라벨 작고 muted.
- **레이아웃**: 흰 상단바, 라운드 18px 카드, 넓은 여백, pill 탭, 미니멀 그림자.
- **원칙**: 과한 애니메이션 금지(`animation:none`), `overflow-y:scroll`로 reflow 방지, 웹폰트 @import 대신 시스템 폰트(이전 깜빡임 교훈 반영).
- **다음**: 다크모드 토글(CSS 변수 기반, zero-flash)로 확장 가능.

## 6. 데이터 한계 (정직성)
무료 데이터로 **불가**한 것은 정직하게 표기/제외한다:
- 실시간 **호가창(L2)**·분봉 실시간 → 불가(유료 피드 필요)
- **실시간 등락 상위 100**(토스 TOP100) → 무료론 제한 → 지수·환율·소규모 큐레이션으로 대체
- **실시간 스크리너** → 일배치 사전계산 + "기준일" 표기
- 애널리스트 컨센서스는 yfinance 제공분만(커버리지 제한) → 출처·한계 명시
