# 🎯 경쟁력·차별화 전략

> "기존 증권사·은행·주식사이트와 무엇이 다른가?"에 대한 답. 정면승부가 아니라 **빈틈(gap) 공략**.

## 0. 냉정한 현실 인식 — 못 이기는 싸움은 피한다
| 상대 | 그들의 강점(우리가 못 이김) |
|---|---|
| 증권사(키움·한투) | 실시간 호가·체결·**주문 실행**, 자체 애널리스트 |
| 토스·뱅크 | UX·사용자 기반·자금력 |
| Bloomberg·Koyfin | 데이터 깊이(유료) |
| ChatGPT 등 범용 AI | 범용 성능 |
→ **정면승부 X.** 위 누구도 잘 못하는 *빈틈*을 노린다.

## 1. 빈틈 분석 — 그들이 약한 곳
- **증권사 앱**: 거래 중심. AI 분석이 약하거나 **블랙박스(근거 없음)**. 리포트가 PDF로 흩어져 검색·종합이 어려움.
- **토스**: 입문자용·단순. **깊은 리서치·근거**가 부족.
- **Yahoo/yfinance**: **영어** 중심. 한국 종목 뉴스·리포트 빈약.
- **범용 AI(ChatGPT)**: **출처 없음·환각**, 실시간 데이터 연결 약함.
> 공통 빈틈 = **"한국어로, 흩어진 정보(리포트·뉴스·지표)를 출처와 함께 종합하는, 환각 없는 AI 리서치"**

## 2. 포지셔닝 (한 문장)
> **증권사 리포트·뉴스·지표를 한국어로 종합해 출처와 함께 답하는, "거짓말 안 하는" AI 리서치 코파일럿.**
- 거래(증권사)도, 입문 UX(토스)도 아닌 **"리서치·판단 보조" 레이어**. 그들과 경쟁이 아니라 *보완*.

## 3. 해자(moat) 3축 — 효중님 강점과 직결
- **A. 정직성·출처 (이미 구축됨 ✅)** — 모든 수치에 도구 근거(EVIDENCE), `as_of` 없으면 날짜 미언급, 없는 종목 거절, 평가 하니스 100%. → **범용 AI의 환각·증권사의 블랙박스와 정면 차별**. 이게 1순위 해자.
- **B. 한국어 도메인 RAG (리포트·뉴스)** — 흩어진 증권사 리포트/뉴스를 모아 "이 종목, 애널리스트들이 뭐라 하나 + 출처". 네이버 뉴스 API(✅) + 공개 리포트. → 증권사 앱에 *없는* "리포트 종합". **효중님의 원래 RAG 프로젝트 목표와 정확히 일치**.
- **C. 도메인 특화 sLLM (EXAONE)** — 한국 금융 용어·감성·요약에 특화. **효중님 EXAONE 파인튜닝 경험 = 차별적 자산**. 범용 GPT 대비 "한국 금융 전용" 스토리.

## 4. 효중님 아이디어 평가 (가치 / 난이도 / 합법성)
| 아이디어 | 가치 | 난이도 | 합법성·리스크 | 판정 |
|---|---|---|---|---|
| **증권사 리포트 크롤링 + RAG** | ★★★ (핵심 해자 B) | 中~高 | ⚠️ 저작권·ToS — **전문 PDF 재배포 금지**. 공개분 메타+요약+출처링크만 | **하되 합법 범위로** |
| **EXAONE 3.5 파인튜닝** | ★★★ (해자 C·ML 깊이) | 高 (GPU·데이터) | ✅ 오픈모델(라이선스 확인) | **단계적으로** |
| **네이버 시세 크롤링** | ★ (이미 yfinance·네이버 펀더 보유) | 低 | ⚠️ 네이버 ToS·robots | **보강용만, 과의존 X** |
| **네이버 뉴스 API** | ★★★ (한국 뉴스·합법) | 低 | ✅ 공식 API | **완료 ✅** |

### 4-1. 리포트 RAG (합법 설계 — 이게 진짜 차별점)
- **수집**: 한경 컨센서스·네이버 금융 '리서치' 등 **공개** 리포트의 *제목·증권사·목표가·요약·원문 링크*만 수집(전문 PDF 재배포 ❌). robots.txt·ToS 준수, 캐싱·요청 제한.
- **파이프라인**: 청킹 → 임베딩(**BGE-m3 무료**) → **Chroma** → 질문 시 관련 리포트 근거 + **출처 링크**와 함께 답변.
- **결과**: "삼성전자에 대한 최근 증권사 목표가·의견 종합" 같은, **증권사 앱·토스에 없는** 기능. 정직성(출처)과도 시너지.

### 4-2. EXAONE 단계 전략
- **1단계(현재·비용0)**: 범용 LLM(GitHub Models)로 품질 확보 ✅.
- **2단계**: **EXAONE-3.5 LoRA 파인튜닝** — (a) 뉴스/리포트 **한국어 감성**, (b) 금융 용어 QA, (c) 리포트 요약. 데이터: 헤드라인+라벨, 공개 리포트 요약쌍. Colab/로컬 GPU.
- **연결**: `LLM_PROVIDER`에 `exaone` 추가(이미 멀티프로바이더 구조 → 확장 쉬움). 포트폴리오 스토리: "교육챗봇 EXAONE 경험 → 금융 도메인 sLLM".

## 5. 차별화 로드맵 (해자 강화 순)
- **P1 — 정직성·한국어 기반 (완료/근접)**: 출처·환각거절 ✅ · 네이버 뉴스 ✅ · 뉴스 AI 감성 ✅
- **P2 — 해자 B 착수**: 공개 **리포트 메타+요약 RAG**("이 종목 애널리스트 의견 종합 + 출처") · 네이버 뉴스 기반 **이슈 타임라인/요약**
- **P3 — 해자 C**: **EXAONE-3.5 LoRA**(한국 금융 감성·요약 특화) → 프로바이더 추가
- **P4 — AI 제품화**: 종목 비교 AI 브리핑 · 포트폴리오 AI 진단 · **출처 포함 한국어 리서치 리포트 자동 생성**

## 6. 네이버 뉴스 API 활용 방안 (지금 적용됨 + 확장)
- **적용 완료 ✅**: 한국 종목 대시보드 뉴스 = 네이버 뉴스(한국어), 해외 = yfinance. 에이전트도 한국 종목엔 `get_naver_news` 사용.
- **확장 아이디어**:
  1. **이슈 타임라인**: `sort=date`로 날짜별 정렬 → 종목별 최근 이슈 흐름.
  2. **뉴스량 급증 감지**: 일별 기사 수 → '관심 급증' 시그널(시세 급변과 결합).
  3. **감성 추세**: 네이버 뉴스 + AI 감성(✅) → 종목 감성 점수 추이.
  4. **검색어 확장**: "삼성전자 실적", "삼성전자 목표주가" 등 의도별 쿼리.
  5. **RAG 보강(P2)**: 뉴스 본문 링크를 리포트 RAG 코퍼스에 함께 투입.

## 6-b. TradingView '세계 시총 랭킹' 대비 차별점 (사용자 관찰 기반)
TradingView 'World's Largest Companies' 페이지의 빈틈 → 우리 차별화:
| TradingView 한계 | 우리 해법(구현됨) |
|---|---|
| 기업 클릭해도 **뭐하는 회사인지 모름** | **기업개요 탭**(섹터·산업·사업요약·직원·본사) + **AI 한국어 종합 브리핑**(출처) |
| 상위 100개 위주 | **국가별 균형 큐레이션**(소국 기업도 포함) — 양보다 국가 다양성·깊이 |
| 데이터만(영어) | **한국어 + 출처 + 정직성**, 랭킹 **클릭 → 차트·AI 분석** 원클릭 |
| 나라 선택 제한적 | **14개국**(미·한·일·중·대만·프랑스·독일·네덜란드·스위스·캐나다·영국·인도·호주·사우디) 나라별/섹터별 비교 + 국가별 시총 합계 |

> 정직한 한계: 무료로 "전 세계 모든 기업"은 불가(유료 데이터 필요). **국가별 대표주 큐레이션 + 깊이(개요·AI·차트)**가 우리 포지션. 종목은 `build_universe.py`로 확장 가능.

## 6-c. 추가 벤치마크(investing·choicestock·wepoll·네이버) & 남은 로드맵
조사: **investing.com**(지수·캘린더·관심목록·스크리너·ProPicks AI·커뮤니티) · **초이스스탁**(스마트스코어 22요인·배당스코어·**적정주가**·AI 매매신호) · **wepoll/네이버 토론**(종목 여론·커뮤니티).

**남은 요청 항목 — 정직한 실현 가능성:**
| 항목 | 가능성 | 경로/한계 |
|---|---|---|
| 분기별 재무 | 🟢 쉬움 | yfinance `quarterly_income_stmt` (다음 바로 가능) |
| **사업부별 매출·영업이익** | 🟡 한국만 | yfinance 미제공 → **DART Open API**(무료, 한국 기업 사업부문) 연동 필요. 해외는 어려움. = **한국 한정 차별점** |
| 현실적 글로벌 시총순위 | 🔴 한계 | 무료로 "전 세계 전 종목" 불가 → 유니버스 확대(배치 무거움) 또는 "표본 기준" 정직 표기 |
| **진짜 리포트 RAG + DB** | 🟡 합법범위 | 전문 PDF 재배포 ❌. 공개 리포트 **메타+요약+링크**만 수집 → BGE 임베딩 → Chroma/SQLite. 별도 스프린트 |
| 종합 점수·적정주가(초이스스탁류) | 🟢 가능 | 보유 데이터(수익률·RSI·PER·애널 상승여력)로 **투자매력 점수** 산출 = 차별점 |
| 커뮤니티 여론(wepoll/네이버) | 🔴 한계 | 사용자 풀 없음 → **뉴스량·AI 감성 추세로 '관심도' 근사**가 현실적 대안 |

## 7. 한 줄 요약
> 못 이기는 곳(거래·실시간데이터·자금)은 피하고, **한국어 · 출처 · 정직 · 도메인LLM**이라는 빈틈에서 **"거짓말 안 하는 한국 금융 AI 리서치"**로 차별화한다. 합법·정직이 곧 브랜드.
